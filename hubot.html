Hubot: Multiplayer FPS

Project Type: First Person Shooter Game
Initial Deployment: May 2017
Latest Update: February 2020

Description

Hubot represents a sophisticated multiplayer First Person Shooter (FPS) meticulously developed within the Unity platform. Originally conceived for an intra-college event hosted under the SmartWit initiative, Hubot boasts the capacity to accommodate up to 20 simultaneous online players while offering a comprehensive array of fundamental FPS functionalities. Leveraging the Unity Editor, the project seamlessly integrates pre-designed and meticulously crafted characters, sculpted using Blender, while the game logic is meticulously implemented in JavaScript, underscoring my proficiency in game development and innovative utilization of industry-standard tools and technologies.
